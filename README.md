# Session-Notation Song Prompting (Cursor Skill)

Dieser Repo enthält einen Cursor Skill, der hilft, **Lyrics + Produktion/Arrangement + Performance-Regie** als „Session-Notation“ zu prompten (Regie-Tags in `[...]`, Lyrics als normale Zeilen), inkl. Bar-/Beat‑Timing, Hit-Points und wiederverwendbaren Motifs.

## Inhalt

- `SKILL.md` – Skill-Definition
- `references/LIBRARY.md` – Bausteine (Copy/Paste)
- `references/EXAMPLES.md` – Beispiele
